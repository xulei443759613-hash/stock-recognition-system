## 短线模式边界

- 只适合训练仓，不适合满仓或重仓。
- 对 34,000 元账户，默认训练仓为 3,400 元，单笔最大计划亏损为 170 元。
- 只有主系统结论为“小仓位试错”且风险收益结构达标时，短线计划才允许真实买入。
- 群消息、截图、老师话术都只作为线索；行情、公告、财报、交易所和可信数据源优先。

## 常用命令

```powershell
python -m compileall stock_recognition_system examples tests
python -m unittest discover -s tests
python -m stock_recognition_system.cli review `
  --message-file examples/group_message.txt `
  --push-date 2026-06-30 `
  --push-time 11:00 `
  --auto-market-data `
  --history-days 20 `
  --account-value 34000
python -m stock_recognition_system.cli outcome `
  --stock-code 603991 `
  --source group `
  --action 放弃 `
  --push-date 2026-06-29 `
  --push-time 14:50 `
  --target-price 138.30 `
  --stop-loss 113.10 `
  --max-price 141.15 `
  --min-price 121.88
python -m stock_recognition_system.cli source-score --source group
```

## 机会评级和错失复盘

- 新增机会评级：真实仓位结论不放松，但报告会区分“可小仓”“模拟跟踪”“等待更优价格”“补证据观察”“剔除机会”。
- 对盈亏比不足的样本，不再只写放弃；会反推普通风控最高买入价、短线最高买入价、100 股止损风险最高买入价和训练模式综合可执行价。
- 东岳硅材样本：21.37 买入不划算，但不是永久删除机会；短线训练综合可执行价为 20.50，需要约 4.07% 回撤后才重新评估。
- `outcome` 和 `source-score` 增加错失机会统计：可执行错失、非可执行上涨、顺序待查。
- 产品原则：放弃后上涨不等于系统错；只有曾回到可执行价且未先触发止损，再触达目标，才算可执行错失。

- 如果当前不允许真实仓位，建议价只用于等待条件和模拟复盘，不作为追入指令。

## 训练档位

- 新增 `training.py`，把严格风控结论进一步翻译成新手可执行的 A/B/C/D 档。
- A 档：严格短线条件通过，只允许训练仓 100 股；B 档：主信号未到 A，但价格、止损、100 股亏损和轻仓盈亏比达标，可做 100 股训练；C 档：模拟观察或到价提醒；D 档：触发硬性否决，放弃。
- 报告新增“训练档位”章节，直接输出是否允许真实 100 股、参考买入价、综合建议止盈价、综合建议止损价、预计盈利、预计亏损和执行清单。
- 东岳硅材 21.37 口径为 C 档：不是删除机会，而是等待 20.50 附近重新评估；高于训练可执行价时不自动买入。
- 新增测试覆盖 A/B/C/D 四类档位，避免系统只会保守放弃，也避免把观察误当成无条件买入。
